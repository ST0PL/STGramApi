# STGramApi
Библиотека для работы с Telegram Bot API, разработанная в рамках индивидуального проекта

## Возможности
- Отправка сообщений
- Разметка текста
- Отправка документов
- Отправка аудиосообщений
- Отправка фотографий
- Редактирование сообщения
- Отправка сообщейний с разметкой reply_markdown
